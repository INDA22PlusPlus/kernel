# Dependencies for MacOS
`qemu-system-x86_64`, `objdump` (llvm package or binutils), `mkfs.fat` (dosfstools)

# tips
Use `make` to compile rust code + assembler bootloader + create the image + qemu run. No need for rust `bootimage`.
